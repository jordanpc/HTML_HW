NBA Evolution - Visualized

How has the impact of player heights/body measurements, positional changes, and rule changes affected the eventual MVP winner of any given NBA season? This was the research question asked prior to obtaining the data, cleaning it, and creating visualizations of the results.

The data was sourced from Basketball-Reference and Basketball Real GM using Python web-scraping tool for reading an html file using the following code: pd.read_html. The data was then converted into a readable CSV file using: pd.read_csv.

The data visualizations and descriptions can be viewed on the Comparisons page, individual Plot pages, and the raw data can be viewed on the Data page.
